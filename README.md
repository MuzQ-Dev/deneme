# Vercel Project - Giriş ve Dashboard Sistemi

Next.js, MySQL ve Cloudinary ile profil yönetim sistemi

## Özellikler

✅ Kullanıcı girişi (MySQL)
✅ Dashboard
✅ İsim, soyisim düzenleme
✅ Profil resmi yükleme (Cloudinary)
✅ Profil resmi silme
✅ Responsive tasarım

## Kurulum

```bash
npm install
```

## Veritabanı Kurulumu

```bash
npm run setup-db
npm run update-table
```

## Çalıştırma

```bash
npm run dev
```

Tarayıcıda: http://localhost:3000
